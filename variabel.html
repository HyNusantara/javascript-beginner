Variabel dan Tipe Data - HyNusantara Academy

Variabel digunakan untuk menyimpan data. Berikut contoh variabel:

        var nilai = 10;
        var nama = "Budi";
        var mhs = {nama: 'Budi', nim: '12345'};

Setiap variabel memiliki tipe data. Tipe data di Javascript antara lain:

  1. Number
  2. String
  3. Object, dsb

Javascript tidak perlu deklarasi tipe data, karena javascript bisa mengubah tipe data otomatis sesuai data yang disimpan.

Perhatikan console, atau tekan F12
