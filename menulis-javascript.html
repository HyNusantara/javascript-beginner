Menulis Javascript Pertama Kali - HyNusantara Academy

Ada 2 cara menulis Javascript, yaitu:

  1. Di dalam file HTML
  2. Di dalam file Javascript (*.js)

Penulisan Javascript di dalam file HTML dapat ditulis pada:

  1. Di dalam tag/elemen head
  2. Di dalam tag/elemen body

Untuk melihat hasil dari Javascript, silahkan tekan tombol F12 dan pilih Tab Console.

Materi selanjutnya adalah Menampilkan Output dari Javascript.
