Menampilkan Output - HyNusantara Academy

Ada 3 cara sederhana menampilkan output dari Javascript:

  1. Console
  2. Elemen
  3. Alert

Console

Gunakan console.log() untuk menampilkan output di console. Akses console pada browser menggunakan tombol F12.

Elemen

Gunakan innerHtml pada setiap object elemen. Maka outputnya langsung dapat dilihat di halaman HTML.

Alert

Gunakan alert() untuk menampilkan output berupa pop-up.

Materi selanjutnya adalah Variabel dan Tipe Data
