Operator pada Javascript - HyNusantara Academy

Operator adalah simbol-simbol yang digunakan untuk melakukan operasi atau proses pada pemrograman

Ada beberapa jenis operator pada Javascript, yaitu:

  1. Assignment
  2. Arithmetic
  3. Comparison
  4. Logical
  5. String and Number
  6. Type
  7. Bitwise

Materi berikutnya adalah If Else.
