Array - HyNusantara Academy

Array adalah bentuk variabel yang dapat menyimpan banyak data dalam satu nama variabel.

Perhatikan contoh koding berikut!

        // Membuat array 
        var arraySatu = [19, 23, 4, 7, 18];

        // Membuat array
        var arrayDua = new Array(19, 23, 4, 7, 18);

Cara menggunakan data yang sudah disimpan di dalam array adalah sebagai berikut

        // Akses data pada array
        console.log( arraySatu[3] ); // 7

Cara mengubah data di dalam array adalah

        // Ubah data pada array
        arraySatu[3] = 15;

Array juga dapat diakses menggunakan for atau while.

Perhatikan koding berikut.

        // Akses array menggunakan for
        for(let i = 0; i < arraySatu.length; i++) {
            console.log( arraySatu[i] )
        }

Materi selanjutnya adalah Function.
